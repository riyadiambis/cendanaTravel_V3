## Langkah Cepat

1. **Buka browser** (Chrome, Firefox, atau Edge)

2. **Akses website** dengan salah satu cara:
   - Jika menggunakan XAMPP: `http://localhost/cendana-travel/index.php`
   - Jika menggunakan server lain: sesuaikan dengan konfigurasi server kamu

3. **Refresh halaman** dengan:
   - Windows: `Ctrl + F5` (hard refresh)
   - Mac: `Cmd + Shift + R`

4. **Lihat perubahan** di halaman beranda:
   - Hero section yang lebih besar dan impactful
   - Typography yang lebih modern dengan font Inter
   - Spacing yang lebih proporsional
   - Card design yang lebih elegan
   - Warna yang lebih profesional

## ğŸ› Jika Ada Masalah

1. **CSS tidak berubah?**
   - Hard refresh: `Ctrl + F5` (Windows) atau `Cmd + Shift + R` (Mac)
   - Clear browser cache
   - Pastikan file `styles.css` sudah tersimpan

2. **Font tidak muncul?**
   - Pastikan koneksi internet aktif (font Inter dari Google Fonts)
   - Refresh halaman beberapa kali

3. **Layout berantakan?**
   - Pastikan tidak ada error di console browser (F12 > Console)
   - Cek apakah semua file CSS ter-load dengan benar
